# Functions-javascript
Functions JavaScript Practise


JavaScript Functions: Reusability and Logic
This script explores different ways to define and execute functions in JavaScript, focusing on parameter handling and return values.

Key Features:
Standard Functions: Demonstrates basic function declaration to eliminate code repetition by passing a string parameter (person) into multiple log statements.

Return Values: Showcases functions that calculate and return data, allowing the result to be stored in variables for later use.

Default Parameters: Includes a function (sum2) that uses a default value for a parameter (z = 100), ensuring the code runs even if one argument is missing.

Arrow Functions: Implements modern ES6 syntax using the const keyword and arrow operator (=>) for concise function definitions.
